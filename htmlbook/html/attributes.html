Atrybuty HTML

Poprzednia Następna

Znaczniki mogą mieć wewnątrz dodatkowy tekst dostarczający kolejnych imformacji o danym znaczniku. Te informacje są nazywane atrybutami i zwykle przyjmują postać pary nazwa=wartość umieszczonych po nazwie znacznika i spacji.

<nazwaznacznika nazwaatrybutu="wartośćatrybutu">
  Dowolny tekst
 </nazwaznacznika>

Znaczniki HTML nie rozróżniają wielkości liter i dlatego można je zapisywać literami dużymi, małymi lub połączeniem obu stylów. Stąd <HTMNL> oznacza dokładnie to samo co <html> i <HtmL>. Atrybuty również nie rozróżniają wielkości liter.

Elementy HTML mogą być modyfikowane za pomocą atrybutów umieszczanych wewnątrz znacznika otwierającego element. Wiele elementów obsługuje specjalizowane atrybuty, a również mamy kilka elementów globalnych możliwych do użycia w dowolnym znaczniku. Na przykład atryut id jest używany w celu wskazania indentyfikatora unikatowo identyfikującego element na stronie. Te identyfikatory są używane wraz z kodem javaScript i kaskadowymi arkuszami styłów.

<h1 id="top-heading">Atrybuty HTML</h1>

Atrybut został umieszczony wewnątrz znacznika otwierającego. Istnieje możliwość umieszczenia wielu atrybutów w pojedynczym znaczniku, jak jest w poniższym przykładzie.

<h1 id="top-heading" class="bold">Atrybuty HTML</h1>

Atrybut class to kolejny atrybut globalny, który może być wykorzystany w celu zdefiniowania dowolnej grupy elementów. Tę samą klasę można przypisać wielu elementom.
